## 🎯 Что это

Готовый скилл для Claude, который переключает ИИ в режим **жёсткого оппонента**. Вместо того чтобы поддакивать и смягчать формулировки, Claude начинает играть роль:

- юриста противоположной стороны
- научного рецензента
- инвестора на Shark Tank в плохом настроении
- редактора-разгромщика

**Задача скилла:** найти и показать всё, что не работает в твоей позиции, пока это не сделал оппонент в суде.

## ⚡ Что делает скилл

Когда ты даёшь Claude документ или позицию и включаешь этот режим, он выдаёт структурированный разбор:

1. **Короткий вердикт** — одним ударом, в чём главная проблема
2. **Главные уязвимости** (3–7 пунктов по убыванию опасности) — с конкретикой: что слабо, почему это проблема, что с этим делать
3. **Атака на сильные стороны** — удар по тому, что ты считаешь своим козырем
4. **Вопросы без ответа** — 5–10 конкретных вопросов, которые задаст оппонент и которые поставят тебя в тупик
5. **Худший сценарий** — как выглядит дело, если всё пойдёт не так
6. **Что спасает** — короткий блок о реальных шансах (только если они есть)

## 🧩 Что включает пакет

```
devils-advocate/
├── SKILL.md                          # Основной файл скилла
├── references/
│   ├── legal-checklist.md            # Чек-лист юридических уязвимостей
│   ├── general-checklist.md          # Чек-лист для идей и решений
│   └── example-responses.md          # Примеры разборов
├── README.md                         # Этот файл
├── INSTALL.md                        # Инструкция по установке
├── DISCLAIMER.md                     # Правовой дисклеймер
└── LICENSE                           # Лицензия MIT
```

**Юридический чек-лист** покрывает:
- Процессуальные уязвимости (сроки, подсудность, надлежащие стороны)
- Материально-правовые (квалификация, причинная связь, размер требований)
- Доказательственные (допустимость, относимость, достоверность)
- Тактические (противоречия, слабые места в показаниях, свежая практика ВС)

**Общий чек-лист** покрывает:
- Логика и аргументация (скрытые допущения, подмена тезиса, ошибка выжившего)
- Фактура (несогласованные цифры, устаревшие данные, выборочное цитирование)
- Реалистичность (ресурсы, зависимости, план Б)
- Конкуренция и последствия второго порядка

## 📥 Как установить

Есть **три способа** — выбери удобный. Подробнее в [INSTALL.md](./INSTALL.md).

### Способ 1: Как user skill в Claude.ai (для Pro/Team/Enterprise)

1. Скачай архив с [последнего релиза](../../releases/latest)
2. В Claude.ai → Settings → Capabilities → Skills → Upload
3. Загружай — скилл начнёт триггериться автоматически по ключевым словам

### Способ 2: Как Claude Project (работает у всех Pro)

1. Создай новый Project в Claude.ai
2. В Custom Instructions вставь содержимое `SKILL.md`
3. В Knowledge добавь файлы из папки `references/`
4. Пользуйся

### Способ 3: Как промпт в любом чате (работает везде)

1. Скопируй содержимое `SKILL.md`
2. Вставь в начало любого нового чата Claude
3. Задавай свои вопросы

## 🚀 Как использовать

Просто напиши Claude одну из фраз:

- «Адвокат дьявола — разнеси мой иск»
- «Найди слабые места в этой позиции»
- «Стресс-тест моего договора»
- «Что спросит суд, чего я не учёл?»
- «Где я неправ?»
- «Играй роль юриста ответчика»

Или приложи документ и напиши: **«devil's advocate»**.
